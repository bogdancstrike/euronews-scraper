# Euronews.ro Scraper

* Scrapes articles from certain category on Euronews.ro
* Articles are automatically categorized in one of the following sentiments: Neutral, Positive, Negative
* The ratio of subjective news / objective news is returned. Quality news are objective.
* Exports articles extracted in file `output.json`

### RUN

```code
python3 .\main.py
```
